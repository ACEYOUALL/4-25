# 勘误表（正片开始，代替答辩吧，我真的很生气🤐）

1. “第四部分” P105：对比结果、分析误差，然而**用的不是本人的精确数据**！而是**图示数据**，仅有一位小数的精度，与两位小数的结果强行对比，**并非数值分析真正意义上的“舍入误差”**
2. “第四部分” P112：**严重误解本人的非线性边界处理**！混淆了**迭代步**和**时间步**，本人使用的 Picard 迭代是**不动点隐式耦合迭代**，显然**不可能导致精度退化**，更不存在“**解耦**”、“**系统性低估**”的可能，无稽之谈
3. “第四部分” P113：**严重误解本人的热量计算**！这是我的计算式吗？**完全无视我后文数值分析部分的验证工作**。COMSOL 不是™圣母，“第三部分”最后用的还不是对热流密度的**离散积分**，这怎么就不提呢？
4. “第四部分” P114：又来“**系统性偏低**”，一边说“**升温前期**”，一边又说“**持续2e6秒**”，前期有那么长吗？“第四部分”是看不见**我估计的离散误差**吗？
5. “第四部分” P116：“**误差抵消**”是什么东西？**哪里有系统误差**？况且我已经验证收敛阶等于 2.0
6. “第四部分” P113：“第二部分”哪里有调用`numpy.linalg.solve`？**虚空索敌**是吧？`1e-13`这个阈值还不够严格吗？
7. “第四部分” P110：积分时间与“第二部分”本人的时间并不一致（因为是**四舍五入差不多手动输入 COMSOL 的**），都不是相同的物理时空。而且“第四部分”的**积分时间步长设置为τ=100秒**，这对结果精确度是致命的负面影响，怎么就不提了呢？
8. “第四部分” P101 和 106：看 COMSOL 导出的温度变化曲线，**前期有明显的“空窗期”和非物理性失真**，这和它的求解机制、非结构化网格特性有关，这显然**影响了初期瞬态热流的积分**，怎么就不提了呢？
